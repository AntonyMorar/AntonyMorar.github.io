PocoCards

A turn base strategy card game made in Unity

About this game

Poco cards is a strategy game inspired by games like Yugioh and Clash Royale, but for this occasion I have tried to make the game juicier and faster with fewer cards to collect.

I've already learned a lot working on this project. I have understood how to make better systems and decouple them from the rest, thanks to this the efficiency and effectiveness in scalability and changes improves considerably.

For more info: itch.io page

Role: Sole developer
Programs: Unity, photoshop, Procreate
Production: 3 Months
Release Date: January 2023

Gameplay

Start with three cards in your hand and strategize to beat your opponent in this turn-based combat game. The deck you equip determines your victory, unlock cards and build the best combination possible.

Deck System

For the system, I designed my own save system and using Scriptable objects I designed the deck system that allows you to have different types of decks within your card collection. With this system it is easy to change the deck of the enemies with a single click.
