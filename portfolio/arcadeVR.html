Arcade VR

A 2 arcade games to play in virtual reality

About this game

This project is an implementation of two games for Oculust Quest. Both games are designed to last 3 minutes each since they will be used within a virtual reality HUB where there is a rotation of attendees.

This game was made for: Inmersys

Role: Programmer
Tools: Unity, Oculust Quest 2, Blender
Team: 5
Production: 3 Months
Status: Full product release

Gameplay

A cyberpunk Box Game where your main objective is to hit the targets depending on the color of your gloves. Make a lot of points and turn off all the batteries!

Sci-fi Firsr person shooter. Kills the robot waves of enemies before time runs out.

CMS Integration

For this project, an integration of a CMS set up by the team is required. To make the connection, the UnityWebRequest module was used. I also implemented part of the leaderboards database in firebase.
